# 서비스 개요

## 한 줄 정의

사주를 처음 접하는 사람도 쉽게 이해하고, 일상에서 활용할 수 있는 사주 해석 서비스

## 서비스 배경

사주 서비스 시장은 이미 존재하지만, 대부분 다음 문제를 가지고 있습니다.

- 한자어·전문 용어 중심으로 일반 유저가 이해하기 어려움
- 불안을 자극하거나 과장된 운명론적 표현이 많음
- 결과를 읽은 뒤 "그래서 어떻게 하라는 건지" 모르는 경우가 많음

이 서비스는 **쉬움 · 신뢰감 · 안도감**을 핵심 원칙으로, 사주를 삶의 참고 데이터로 활용할 수 있도록 돕습니다.

## 타겟 유저

| 유형 | 설명 |
|------|------|
| 사주 입문자 | 사주에 관심은 있지만 어디서 어떻게 봐야 할지 모르는 20~30대 |
| 가벼운 탐색 유저 | 재미·호기심으로 자신의 기질과 흐름을 확인하고 싶은 유저 |
| 의사결정 참고 유저 | 취업·연애·이사 등 주요 결정 시 사주를 보조 데이터로 활용하고 싶은 유저 |

## 핵심 기능

### 1. 만세력 조회 (무료)
- 생년월일시 입력 → 사주팔자(년·월·일·시주) 도출
- 기본 오행·음양 구성 확인
- 일주(日柱) 기반 기질 요약 제공

### 2. 상세 사주 해석 (유료 · 10,000원)
- 용신·격국 기반 심층 분석
- 대운·세운 흐름 해석
- 관계운·직업운·재물운 영역별 해석
- 전문 용어 사용 시 반드시 쉬운 언어로 병기
- 솔루션 중심 조언 포함

## 서비스 플로우

```
[입력]
이름 · 성별 · 생년월일 · 태어난 시 · 태어난 도시

       ↓

[만세력 조회] (무료)
사주팔자 구성 확인
기본 기질 요약

       ↓

[상세 조회 유도]
"더 자세한 해석을 원하시나요?" → 10,000원 결제

       ↓

[상세 해석 결과]
영역별 분석 + 솔루션 제공
```

## 비즈니스 모델

| 항목 | 내용 |
|------|------|
| 무료 기능 | 만세력 조회, 기본 기질 확인 |
| 유료 기능 | 상세 사주 해석 |
| 가격 | 1회 10,000원 (건당 과금) |
| 추후 고려 | 구독 모델, 궁합 조회, 신년운세 패키지 |
